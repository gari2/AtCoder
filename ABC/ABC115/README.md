# [AtCoder Beginner Contest 115](https://atcoder.jp/contests/abc115)  
  
### A - Christmas Eve Eve Eve  
[問題ページ](https://atcoder.jp/contests/abc115/tasks/abc115_a)  
- #### 解法  
    `Christmas`という文字列に 25 - D 個 (半角空白 + `Eve`)を追加  
  
- #### コード()  
  
    ```python
    D = int(input())

    ans = 'Christmas'
    for i in range(25 - D):
        ans += ' Eve'

    print(ans)
    ```
